# Data layout

## Raw data

These raw data files are stored locally and are ignored by git.

- `data/raw/ARC/evaluation/`  
  ARC official evaluation JSON files

- `data/raw/H-ARC/`  
  H-ARC csv files, including `incorrect_submissions.csv` and `summary_data.csv`

- `data/raw/V-ARC/VARC_predictions/`  
  VARC prediction files

## Notes

- Raw data should not be committed to GitHub.
- Only scripts, documentation, and processed data should be tracked by git.
